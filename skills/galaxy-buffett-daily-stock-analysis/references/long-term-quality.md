# 长期质量评估框架

**何时读取：** 仅在月度复盘，或可验证的新事实可能改变某项实际持仓的投资逻辑时读取。

**何时不要读取：** 日常新闻扫描、短期价格波动、静态估值变化，或投资逻辑没有变化时不要读取。

## 评估六项长期要素

1. **护城河**：识别品牌、网络效应、转换成本、规模、成本优势、数据或监管壁垒；用份额、留存、定价权或单位经济性验证其是否扩大或收窄。
2. **资本回报**：观察 ROIC、增量资本回报、自由现金流转换和再投资空间；区分经营改善、资本结构变化与周期性高点。
3. **资产负债表**：检查净债务、利息覆盖、债务期限、流动性、表外义务和稀释风险；评估下行情景中的生存与再投资能力。
4. **管理层**：评估资本配置、指引可信度、激励一致性、治理和历史执行；用长期记录而非单次发言判断。
5. **估值区间**：按保守、基准、乐观情景给出关键经营假设与合理区间；显式说明增长、利润率、资本成本和终值敏感性。
6. **反证**：列出能够推翻投资逻辑的可观察指标、阈值和检查期限；主动寻找竞争、监管、技术替代与资本配置失败证据。

## 结论纪律

- 遵守固定新闻证据 reference 已建立的证据门槛；以长期可比数据和一手披露支持判断，口径变化必须披露。
- 先判断企业质量和投资逻辑，再判断价格；不得用低市盈率、低市净率或较历史均值折价等静态低估值代替质量。
- 将结论写成“逻辑加强 / 逻辑不变 / 逻辑弱化 / 逻辑破坏”，并说明变化来自哪项长期要素。
- 只有投资逻辑或安全边际发生可验证的实质变化时才提出条件式动作；每项动作写明触发条件、反证条件、时间范围和主要风险。
- 月度复盘没有新反证时写“维持原计划”，不要为形成观点而制造交易。
