# 财报分析框架

**何时读取：** 实际持仓公司临近财报，或刚发布财报并命中当日扫描时读取。

**何时不要读取：** 没有持仓财报命中、只有泛行业财报热度，或已经完成一次财报额外深挖时不要读取。

## 先分开四类事实

1. **已发生结果**：记录报告期、收入、利润率、现金流和分部表现，不把历史结果写成未来趋势。
2. **未来指引**：记录管理层给出的区间、期间、假设和口径；没有正式指引时明确写“未提供”。
3. **市场预期**：注明一致预期、发布时间和来源，分别计算实际结果及指引相对预期的差异。
4. **价格反应**：记录盘前、盘中或盘后的窗口与幅度；将其视为市场反馈，不视为基本面证据或交易理由。

## 执行比较

- **收入**：比较总收入、同比/环比、内生增长、汇率影响和关键分部；区分一次性项目与持续性需求。
- **利润率**：分别比较毛利率、营业利润率和净利率；标明 GAAP / non-GAAP，并解释重组、股权激励等调节项。
- **现金流**：比较经营现金流、资本开支、自由现金流和自由现金流转换率；指出营运资本拉动或拖累。
- **指引**：比较新指引、上一版指引和市场预期；记录上调、维持、下调及区间宽度变化。
- **管理层措辞**：对照本期与上期正式材料，标记需求、定价、库存、产能和成本措辞的强化、弱化或新增限定。
- **口径**：统一币种、会计口径、持续经营范围和每股基准；口径不可比时不要强算预期差。

## 证据与深挖纪律

- 遵守固定新闻证据 reference 已建立的证据门槛；用 SEC 文件或公司官方 IR 材料为财报数字、正式指引和管理层表述定案。
- 独立媒体用于确认市场预期、价格反应和外部解读；数字冲突时以最新一手材料为准并披露差异。
- 先完成全部持仓财报扫描，再按“仓位影响、预期差、投资逻辑相关性”选择最多一个命中执行额外深挖。
- 额外深挖只解决一个关键不确定性；不得因多个财报命中增加第二次专项分析。

## 绑定投资逻辑与动作

先判断原投资逻辑为**未改变 / 需要观察 / 已被破坏**，再考虑动作。只把可验证且能够连接到收入质量、利润结构、现金创造或未来竞争力的变化写入动作依据。

每项动作必须写明：触发条件、反证条件、时间范围和主要风险。单次超预期、单季成本波动、管理层乐观措辞或价格反应不得单独触发交易；投资逻辑未被破坏且无新证据时写“维持原计划”。
